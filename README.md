# ZoneAnomaly

This is a project for https://github.com/yakhira/ZoneRadar. 
Simulates zone for Stalker RPG, contains PIR and Sound sensor, works as BLE devices. 
Components:
- attiny 13a (https://www.aliexpress.com/item/1005004884427971.html)
- JDY-19 (https://www.aliexpress.com/item/32911430913.html)
- AM312 PIR sensor 
- KY-037 Sound Detection Sensor (https://www.aliexpress.com/item/1005001621954302.html)
